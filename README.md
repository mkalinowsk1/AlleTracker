# AlleTracker
**AlleTracker** is a graphical data representation tool for product information, utilizing the Allegro API. It allows users to visualize key product data in an intuitive, user-friendly interface.
## Introduction
AlleTracker is currently being developed by a team of students from the IT Institute at the University of Gda≈Ñsk as part of a class project. However, there are plans for it to evolve into a standalone tool that will be available for everyone to use.
